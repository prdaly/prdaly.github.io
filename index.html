Automated Quadcopter

May 2021 - Present I am currently developing an automated quadcopter, which will be equipped with sensors and instruments to be controlled using a RF receiver. My main goal in creating this project is to experiment with PID control, Arduino, and using sensors (GPS receiver, barometer, accelerometer, gyro) to allow a drone to autonomously hover in one position. I have successfully created a 2D self-balancing mockup, with two brushless motors on either end of a beam. Using measurements from an accelerometer and gyroscope, the motors can self-level using Kalman Filters and PID control loops. I am currently working on connecting a receiver to control motor thrust with joysticks, as well as prototyping 3D printed frame parts. Once this frame is made, I will be able to expand my experiment to a 3D self-leveling drone with 4 brushless motors.
